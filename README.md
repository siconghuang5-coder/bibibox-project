# BibiBox 核心引擎与数字人终端架构

BibiBox 是一款集成了**大模型神经认知层 (基于 Coze AI V3 API)** 与**前卫全息礼物交互协议**的新生代微信小程序端与后端集成平台。

本仓库包含了本项目的全链路源代码，包含 NestJS 核心大脑与基于 Uniapp 的微信客户端基座。目前，底层核心的重型预置图片资源已整体上移并架设至全球 GitHub 高速 CDN 节点（jsDelivr）实现秒开与物理空间剥离解耦。

---

## ⚡ 极简上手测试指南（Leader 专用）

为极大降低环境部署门槛，已在项目根目录下注入「**一体化启动脚本**」。

### 1. 启动全链路基座
项目根目录下包含名为 `一键启动-BibiBox环境.bat` 的程序。
双击它，它将自动化执行：
1. 检测并抹除可能卡死的历史服务端口；
2. 自动化启动 Node/Nest 后端神经服务与路由终端。

*(看到绿幕提示后，后端便已接管底层，请让弹框的 CMD 背景挂在后台即可！)*

### 2. 挂载微信开发者工具
1. 打开您的 **HBuilderX** 编辑器软件。
2. 将本工程目录下的 `bibi-box (2)` 文件夹拖拽进左侧的项目侧边栏。
3. 点击上方菜单栏的：**运行** $\rightarrow$ **运行到小程序模拟器** $\rightarrow$ **微信开发者工具**。

✅ **沉浸体验提示**：
1. 打开小程序后，随便挑一个模型（例如蔡蔡风尚、墨白或柳如烟）。
2. 在左侧聊天输入侧栏点击『送礼』图标，即可体验**无延时流光特效的顶级礼物展现交互系统**！
