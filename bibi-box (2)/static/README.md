# 图标 SVG 占位说明

请将以下 SVG 文件放入本目录，命名与下表一致。图标语义与 Material Symbols / 常见 UI 一致，可从 Material Icons 或类似库导出为 SVG。

| smart-toy.svg | 登录页 Logo、商品占位、AI 角色 |
| wechat.svg | 微信登录按钮 |
| alipay.svg | 支付宝登录按钮 |
| wallet.svg | MetaMask/钱包、个人页余额 |
| link.svg | WalletConnect 登录 |
| search.svg | 搜索 |
| shopping-cart.svg | 市场页购物车 |
| arrow-forward.svg | 查看更多/全部 |
| gift.svg | 礼物模块 |
| favorite.svg | 未点赞 |
| favorite-filled.svg | 已点赞（实心） |
| stars.svg | 星星/VIP |
| add.svg | 加购 |
| add-circle.svg | 聊天/社交页添加 |
| storefront.svg | 商城 Tab（未选） |
| tab-storefront.svg | TabBar 市场（未选） |
| tab-storefront-active.svg | TabBar 市场（选中） |
| tab-groups.svg | TabBar AI社交（未选） |
| tab-groups-active.svg | TabBar AI社交（选中） |
| tab-chat.svg | TabBar 聊天（未选） |

| info.svg | 关于 |

说明：部分小程序/App 的 tabBar 仅支持 PNG，若运行时报错请将 tab-storefront / tab-groups / tab-chat / tab-person 及对应 -active 改为同尺寸 PNG 放在本目录，并在 pages.json 中把 iconPath/selectedIconPath 改为 .png。
